# Expense Tracker

A simple command line expense tracker built in Python.

## What it does

This script lets you enter your expenses one by one. Type "quit" whenever you want to stop, and it will show your total expenses.

## How it works

The program uses a while loop to keep asking for expenses until you type "quit". It uses try and except to catch invalid input, so if you type something that is not a number, it will ask you to enter a number again instead of crashing.

## How to run
python expTracker.py

## Example
your expense tracker:
enter your expense or quit for ending: 200
enter your expense or quit for ending: 150
enter your expense or quit for ending: abc
invalid input, enter number
enter your expense or quit for ending: quit
total: 350

## Concepts used

* While loops
* Try and except for error handling
* Input validation
* Basic arithmetic with variables
